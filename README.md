# Examen-Unidad-2-Grupo-J
En este espacio, se realizo el código del siguiente enunciado propuesto por el docente:
Grupo J. Restaurante universitario con pedidos y tiempos de preparación 

Contexto del caso: Manejar platos, pedidos, ingredientes, tiempos de preparación, retrasos y facturación. 

Clases mínimas sugeridas: Plato y Pedido 

Requerimientos funcionales obligatorios 

Registrar platos con código, categoría, costo, precio, tiempo estimado y disponibilidad. 

Registrar pedidos con cliente, mesa, hora, prioridad y detalle de platos. 

Calcular subtotal, descuento, recargo por urgencia e IVA. 

Buscar por cliente y código, ordenar manualmente por mayor demanda y tiempo de espera. 

Actualizar stock básico de ingredientes y emitir alertas por retraso. 

Integración obligatoria con Ahorcado: Construir un Ahorcado con platos, ingredientes, procesos, pedidos y tiempos. 

- Además se encuentra resuelto un pequeño fragmento solicitado por el docente este se encuentra como (Ejercicio Depuracion)
